# Knowva - MVP開発タスク

## MVP開発のアクションプラン（タスクリスト）

1.  **設計フェーズ (1日)**
    *   [ ] 上記要件に基づいた画面ワイヤーフレーム（主要画面のみ）を作成する。
    *   [ ] データ構造を設計する（シンプルな構造から開始）。
    *   [x] 技術スタック（言語、フレームワーク、DB、LLM API）を選定する。
        * バックエンド: Python (FastAPI)
        * インフラ: Google Cloud Agent Engine
        * Agent Framework: ADK
        * DB: Firestore + Google Cloud Storage
    *   [ ] AIエージェントの対話フローとプロンプト設計を行う。
2.  **開発フェーズ (4-5日)**
    *   [ ] ユーザー認証機能（シンプルな実装）とDBのセットアップを行う。
    *   [ ] 読書記録・対話履歴のデータ保存・表示機能（基本的なCRUD）を実装する。
    *   [ ] LLM API（例: OpenAI API）と連携し、「対話的振り返り」「学びの抽出」のプロンプトを設計・実装する。
    *   [ ] 対話UI（テキスト入力のチャット形式）を実装する。
    *   [ ] 簡易的な本の推薦機能を実装する。
3.  **テスト・デプロイフェーズ (1日)**
    *   [ ] 全体の動作確認とバグ修正を行う。
    *   [ ] テスト環境にデプロイし、関係者で実際に使ってみる。
